This project is an interactive E-commerce Sales Analysis Dashboard built in Microsoft Excel. It provides key business insights by analyzing sales, profit, orders, quantity, and category performance using interactive visualizations and filters.

✨ Features
📈 Sales, Profit, Quantity, Orders & Profit Margin KPIs
📊 Year-over-Year (YoY) Growth Analysis
📅 Monthly Sales vs Profit Analysis
🛍️ Category-wise Profit Analysis
🥧 Sales Share by Category
🗺️ Sales by State Map
🏆 Top 10 Sub-Categories by Sales
🎛️ Interactive Slicers (Year, Region, Segment)
🛠️ Tools & Skills Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Map Charts
Dashboard Design
Data Visualization
